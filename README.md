# Famiq - Kelola Keuangan Jadi Lebih Bermakna 🚀

**Famiq** (Family & Unique) lahir dari sebuah keyakinan sederhana: bahwa **setiap keluarga itu unik**. Begitu juga dengan cara mereka mengelola uang. Ada keluarga yang sangat mendetail, ada yang santai, namun semuanya memiliki tujuan yang sama—membangun masa depan yang lebih baik bersama pasangan.

Kami memahami bahwa mencatat keuangan seringkali terasa seperti beban atau sumber drama. Itulah kenapa **Famiq** hadir untuk merayakan keunikan cara Anda mengelola uang, menyederhanakannya tanpa drama, dan mengubah setiap catatan pengeluaran menjadi langkah menuju keterbukaan dan transparansi.

> **"Pencatatan Cerdas untuk Keluarga Harmonis"**

## ✨ Kenapa Famiq?

- **Rayakan Keunikan Keluarga**: Apapun gaya mencatat Anda dan pasangan, Famiq menyesuaikan dengan alur hidup Anda. Tidak perlu spreadsheet yang kaku dan membosankan.
- **Transparansi Tanpa Drama**: Dengan sinkronisasi real-time, Anda dan pasangan selalu berada di halaman yang sama. Kejujuran finansial dimulai dari sini.
- **Sederhana & Manusiawi**: Kami membuang semua fitur rumit yang tidak Anda butuhkan. Fokus kami adalah membuat pengalaman mencatat uang menjadi sealami mungkin.

## 🚀 Cara Menjalankan Project

Untuk menjalankan project ini di lingkungan pengembangan lokal:

1. **Clone Repository**:
   ```bash
   git clone https://github.com/username/famiq-app.git
   ```

2. **Setup Lingkungan**:
   Project ini memerlukan beberapa file konfigurasi layanan cloud dan API keys yang sengaja kami sembunyikan demi keamanan. Silakan hubungi pengembang untuk mendapatkan panduan struktur file konfigurasi yang diperlukan.

3. **Build & Run**:
   Buka project di Android Studio, tunggu Gradle Sync selesai, dan jalankan di perangkat atau emulator pilihan Anda.
